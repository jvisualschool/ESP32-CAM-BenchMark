# ESP32-CAM CameraWebServer 설정 가이드

> OV2640 카메라 기준 | Advanced Settings 제외

---

## 📷 화질 기본 설정

### Resolution (해상도)
카메라가 촬영하는 이미지의 크기입니다. 해상도가 높을수록 화질은 좋지만 FPS가 낮아집니다.

| 값 | 해상도 | 용도 |
|---|---|---|
| QQVGA | 160×120 | 초저사양, 최고 FPS |
| QVGA | 320×240 | 일반 스트리밍 기본값 |
| VGA | 640×480 | 균형 잡힌 화질/FPS |
| SVGA | 800×600 | 고화질 스트리밍 |
| HD | 1280×720 | 와이드 고화질 |
| UXGA | 1600×1200 | 최고 화질 (PSRAM 필요) |

> PSRAM이 있는 AI-Thinker 보드는 UXGA까지 안정적으로 사용 가능합니다.

---

### Quality (JPEG 압축 품질)
이미지 JPEG 압축률을 조절합니다. **숫자가 낮을수록 고화질**입니다.

- 범위: **4 ~ 63**
- 낮은 값 (4~10): 고화질, 파일 크기 큼, FPS 낮아짐
- 높은 값 (50~63): 저화질, 파일 크기 작음, FPS 높아짐
- **권장값: 10~15**

---

### XCLK (클럭 주파수)
ESP32가 카메라 센서에 공급하는 클럭 속도(MHz)입니다.

- 범위: **4 ~ 40 MHz**
- 높을수록 FPS 향상 가능하나 불안정해질 수 있음
- **권장값: 15~20 MHz** (보드 품질에 따라 15 MHz가 더 안정적인 경우 있음)

---

## 🎨 화면 조정

### Brightness (밝기)
영상의 전체적인 밝기를 조절합니다.
- 범위: **-2 ~ 2**
- 0이 기본값, 양수는 밝게, 음수는 어둡게

### Contrast (대비)
밝은 부분과 어두운 부분의 차이를 조절합니다.
- 범위: **-2 ~ 2**
- 높을수록 명암이 뚜렷해짐

### Saturation (채도)
색의 진하기를 조절합니다.
- 범위: **-2 ~ 2**
- 높을수록 색이 선명하고 진해짐, 낮을수록 흑백에 가까워짐

---

## ✨ 특수 효과

### Special Effect (특수 효과)
영상에 필터 효과를 적용합니다.

| 값 | 효과 |
|---|---|
| No Effect | 효과 없음 (기본값) |
| Negative | 색상 반전 |
| Grayscale | 흑백 |
| Red Tint | 붉은 색조 |
| Green Tint | 초록 색조 |
| Blue Tint | 파란 색조 |
| Sepia | 세피아 (빈티지 느낌) |

---

## ⚪ 화이트 밸런스 (AWB)

### AWB (Auto White Balance)
자동으로 색 온도를 보정해 자연스러운 색상을 만듭니다.
- **ON 권장** (실내외 환경 변화에 자동 대응)

### AWB Gain
AWB 보정 강도를 조절합니다.
- AWB가 ON일 때 활성화
- ON: 더 강하게 보정, OFF: 보정 약하게

### WB Mode (화이트 밸런스 모드)
AWB Gain이 ON일 때 조명 환경을 직접 지정합니다.

| 값 | 환경 |
|---|---|
| Auto | 자동 감지 (기본값) |
| Sunny | 맑은 야외 |
| Cloudy | 흐린 야외 |
| Office | 형광등 사무실 |
| Home | 가정용 백열등 |

## 💡 노출 설정 (AEC)

### AEC (Auto Exposure Control)
카메라가 자동으로 노출(셔터 속도)을 조절합니다.
- **ON 권장** (밝기 변화에 자동 대응)

### AEC2 (야간 모드)
어두운 환경에서 노출 시간을 더 길게 허용합니다.
- 어두운 곳에서 ON하면 더 밝은 영상을 얻을 수 있음
- 단, 빠른 움직임이 흐릿하게 보일 수 있음

### AE Level (노출 보정)
자동 노출의 기준 밝기를 조절합니다.
- 범위: **-2 ~ 2**
- 양수: 전체적으로 밝게 노출, 음수: 어둡게 노출

### AEC Value (수동 노출값)
AEC를 OFF했을 때 노출 시간을 직접 설정합니다.
- 범위: **0 ~ 1200**
- 높을수록 더 밝음 (셔터 느려짐)

---

## 📈 게인 설정 (AGC)

### AGC (Auto Gain Control)
카메라 센서의 신호 증폭을 자동으로 조절합니다.
- **ON 권장** (어두운 환경에서 자동으로 밝게)

### AGC Gain (수동 게인값)
AGC를 OFF했을 때 증폭 강도를 직접 설정합니다.
- 범위: **0 ~ 30**
- 높을수록 밝아지나 노이즈(화면 잡음)도 증가

### Gain Ceiling (최대 게인 한도)
AGC가 자동으로 올릴 수 있는 최대 게인 상한선입니다.
- 범위: **0 ~ 6** (2배 ~ 128배)
- 너무 높이면 어두운 환경에서 노이즈가 심해짐
- **권장값: 2~4**

## 🔧 화질 보정

### BPC (Black Pixel Correction)
센서의 결함 픽셀(검은 점)을 자동으로 보정합니다.
- **ON 권장**

### WPC (White Pixel Correction)
센서의 결함 픽셀(흰 점)을 자동으로 보정합니다.
- **ON 권장**

### Raw GMA (감마 보정)
밝기의 비선형 보정을 적용합니다. 어두운 영역의 디테일을 살려줍니다.
- **ON 권장** (더 자연스러운 명암 표현)

### LENC (렌즈 보정)
렌즈 주변부의 밝기 감소(비네팅)를 자동으로 보정합니다.
- **ON 권장** (화면 가장자리가 더 균일해짐)

### DCW (Downsize EN)
해상도를 낮출 때 품질 손실을 줄이는 다운스케일링 방식을 적용합니다.
- **ON 권장** (저해상도 설정 시 화질 향상)

---

## 🔄 화면 방향

### H-Mirror (좌우 반전)
영상을 좌우로 뒤집습니다.
- 카메라를 거울처럼 사용할 때 또는 설치 방향에 따라 ON

### V-Flip (상하 반전)
영상을 상하로 뒤집습니다.
- 카메라를 거꾸로 설치했을 때 ON

---

## 🧪 기타

### Colorbar (컬러바 테스트 패턴)
카메라 영상 대신 색상 테스트 패턴을 출력합니다.
- **일반 사용 시 반드시 OFF**
- 카메라 정상 작동 여부 확인용 디버그 기능

### LED Intensity (플래시 LED 밝기)
내장 플래시 LED의 밝기를 조절합니다.
- 범위: **0 ~ 255**
- 0: 꺼짐, 255: 최대 밝기
- 스트리밍 중 자동으로 켜짐
- **주의: 최대값은 매우 밝으므로 눈을 직접 바라보지 마세요**

## ✅ 권장 기본 설정

| 항목 | 권장값 |
|---|---|
| Resolution | SVGA 또는 VGA (스트리밍) |
| Quality | 10 ~ 15 |
| XCLK | 15 ~ 20 MHz |
| AWB / AWB Gain | ON |
| AEC / AEC2 | ON |
| AGC | ON |
| BPC / WPC | ON |
| Raw GMA / LENC / DCW | ON |
| Colorbar | **OFF** |
| LED Intensity | 0 (필요시 조절) |
